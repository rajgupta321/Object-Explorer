# Nature Explorer

Nature Explorer is a Flutter-based learning activity app designed for children aged 3–8 years.

The app encourages children to explore their surroundings by finding and capturing pictures of different objects using the device camera.

## Features

- Camera Integration
- Object Recognition using Google ML Kit
- Interactive Learning Challenge
- Reward System for Motivation
- Child-Friendly User Interface
- Progress Tracking

## Challenge Items

- Plant
- Mobile
- Paper
- Television
- Person

## Tech Stack

- Flutter
- Riverpod
- GoRouter
- Google ML Kit
- Image Picker

## How It Works

1. Select a challenge.
2. Capture an image using the device camera.
3. The app identifies the object.
4. Correct detections complete challenge items.
5. Children earn rewards and complete the challenge.
